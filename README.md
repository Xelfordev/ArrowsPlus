Arrows+ adds explosive arrows, spectral tracking, teleport arrows, grappling hooks, and 40+ enchantments for bows and crossbows. Originally created for Bukkit/Spigot, now rebuilt for Paper 1.21.1.

## Features

- **Explosive Arrows** — arrows that detonate on impact (configurable radius and damage)
- **Spectral Tracking** — arrows that reveal and mark targets
- **Teleport Arrows** — teleport to where your arrow lands
- **Grappling Hook** — arrow pulls you to the impact point
- **40+ Enchantments** — frost, lightning, poison, wither, flame, explosive, and more
- **Custom Crossbow Enchants** — multishot enhancements, piercing upgrades
- **Fully Configurable** — all effects, damage values, and probabilities in config

## Installation

1. Download the latest JAR from [CurseForge](https://www.curseforge.com/minecraft/bukkit-plugins/arrows)
2. Place it in your server's `plugins/` folder
3. Restart your server

## Building

```bash
gradlew build
```

Output: `build/libs/ArrowsPlus-2.0.0.jar`
